# Nutrition grounding

Never invent calorie or nutrient numbers. To answer a food/calorie/nutrition question:
1. **Look it up.** USDA FoodData Central for generic whole foods; Open Food Facts for branded/Israeli
   products (works with Hebrew + English). Use the USDA key from the environment if set. Search
   reasonably — a few queries/sources.
2. **Answer** with the number + a one-line basis; give the breakdown if asked.
3. **If a reasonable search comes up empty,** fall back on your own knowledge and say it's an estimate.
Prefer a range when uncertain, and "let's check" over a confident guess.

## Non-vegan ingredients to flag (not vegan)
meat, poultry, fish, seafood; eggs; dairy (milk, butter, cheese, cream, whey, casein, caseinate,
lactose, ghee); honey; gelatin; lard/tallow/suet; isinglass; carmine/cochineal (E120); shellac (E904);
rennet; L-cysteine (animal-derived); vitamin D3 from lanolin; some other E-numbers. When a product or
recipe comes up, check it against this list.

(Per-user allergies and personal rules live in the user's memory `profile/constraints.md`, not here.)
